# Australia Family Trip · 7–19 July 2026

Interactive itinerary for a family trip from Hanoi to Sydney & Melbourne, built around the First Lego League (FLL) tournament at Macquarie University's Wattamlagul Campus.

**Family:** Thai + wife + son (11, FLL competitor) + daughter (9)
**Route:** Hanoi → Sydney (Jul 7–13) → Melbourne (Jul 13–19) → Hanoi
**Airline:** Vietnam Airlines (VN)
**Sydney base:** Oaks Sydney North Ryde Suites

## What's inside

- **Overview** — daily summary and key dates
- **Visa** — Australian subclass 600 tourist visa checklist and timeline
- **Flights** — Vietnam Airlines outbound / return with economy & premium economy fares in AUD and VND
- **Transport** — Uber / Opal / Myki / day tour guidance
- **Hotels** — Oaks Sydney North Ryde Suites (5 nights) + 3 Melbourne CBD options (6 nights)
- **13 day tabs** (Jul 7 – Jul 19) with hour-by-hour timelines
- **Dining** — 3 options per meal with cuisine filter pills (Japanese, Vietnamese, Italian, Seafood, etc.)
- **Budget** — full family cost estimate, mid-range vs premium
- **Packing** — FLL-specific, kids extras, winter layering, documents

Every address links to Google Maps. Every price has a VND equivalent (1 AUD ≈ 16,200 VND).
